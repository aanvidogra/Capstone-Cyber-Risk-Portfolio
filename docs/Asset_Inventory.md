# Asset Inventory

The asset inventory identifies the systems and data that are critical to business operations.

| Asset | Type | Description | Sensitivity | Criticality |
|------|------|-------------|-------------|-------------|
Billing Platform | Application | Core system used for processing medical billing transactions | High | High
Customer Portal | Application | Web interface used by clinics to access billing and reporting | High | High
Patient Billing Data | Data | Contains sensitive billing and financial records | High | High
Payment Gateway | Third Party Service | Processes financial transactions | High | High
Microsoft 365 | SaaS | Email, documents, collaboration tools | Medium | High
Employee Laptops | Hardware | Endpoints used by staff for daily work | Medium | High
Identity Management System | Security | Controls authentication and access | High | High
Backup System | Recovery | Stores system backups for disaster recovery | High | High
Security Monitoring System | Security | Logs and monitors security events | Medium | Medium
Vulnerability Management Tool | Security | Identifies security weaknesses | Medium | Medium

## Key Business Dependencies

The organization depends heavily on the availability and security of its digital infrastructure. Loss of these systems could result in:

- financial loss
- operational downtime
- regulatory violations
- reputational damage
