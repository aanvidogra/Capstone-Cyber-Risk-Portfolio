# NIST Cybersecurity Framework Mapping

The NIST Cybersecurity Framework (CSF) provides a structure for managing cybersecurity risk.

The framework consists of five core functions.

## Identify

Identify organizational assets, systems, and risks.

Examples:
- Maintain asset inventory
- Identify critical business services
- Document third party dependencies
- Perform risk assessments

## Protect

Protect systems and data from potential threats.

Examples:
- Multi-factor authentication
- access control
- endpoint security
- employee security awareness training
- encryption of sensitive data

## Detect

Detect cybersecurity incidents quickly.

Examples:
- centralized log monitoring
- SIEM alerts
- vulnerability scanning
- suspicious login monitoring

## Respond

Respond to security incidents to limit damage.

Examples:
- incident response plan
- security escalation procedures
- containment of compromised systems
- communication with stakeholders

## Recover

Restore systems and operations after an incident.

Examples:
- backup restoration
- disaster recovery planning
- system rebuild procedures
- lessons learned reviews
